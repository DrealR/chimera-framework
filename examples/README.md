## How to Read a Scan

Every scan follows the same protocol:
1. **Identity** — what is this body?
2. **Nine Questions** — the foundational reading
3. **Bump Detection** — what's blocked?
4. **Skeleton** — one sentence of irreducible truth
5. **DNA Layer** — O>I / I>O / Mixed / Conditional reading
6. **Advanced Diagnostics** — five god powers, prime, federation/dominion, surfaces, attention
7. **Structural Weakness** — honest limitations and inversion conditions
8. **Cross-Domain Connections** — where this body's patterns appear elsewhere
9. **Fruit** — what new understanding emerged
10. **Powers Detected** — named capabilities
