## Cross-Domain Patterns

Biology scans surface patterns visible across all domains:
- **Membrane as identity** (cell) → what you let through defines what you are
- **Self/not-self discrimination** (immune system) → authentication, moderation, trust filters
- **Federation at scale** (microbiome) → distributed systems, open-source communities
- **Autoimmune pathology** (immune system) → when protection attacks the body it protects
