## Cross-Domain Patterns

Biology scans surface patterns visible across all domains:
- **Membrane as identity** (cell, neuron) → what you let through defines what you are
- **Self/not-self discrimination** (immune system, cancer) → authentication, insider threat, trust erosion
- **Federation at scale** (microbiome, cell organelles) → distributed systems, open-source communities
- **Autoimmune pathology** (immune system) → when protection attacks the body it protects
- **O>I requires rhythm** (heart) → continuous output without pause is lethal, not generous
- **Fidelity-speed tradeoff** (DNA replication) → accuracy and throughput cannot both be maximized
- **Legacy code** (photosynthesis/RuBisCO) → enzymes optimized for dead atmospheres, too integrated to replace
- **Insider threat** (cancer) → native bodies inverting from serving to extracting using the system's own credentials
