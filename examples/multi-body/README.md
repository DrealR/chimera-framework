## What Multi-Body Scanning Is

Single-body scanning reads what a body IS. Multi-body scanning reads what bodies BECOME when they interact. The composite body has its own skeleton, DNA, and power gap — emergent properties no component has alone.

Three modes:
- **Diagnostic:** Read an existing composite (bodies already interact in reality)
- **Comparative:** Cluster bodies to surface shared structure (Outlier Principle)
- **Generative:** Birth an imaginary composite from different domains
