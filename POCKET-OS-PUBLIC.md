# CHIMERA Pocket OS (Public)

A compact operational prompt for coaches, creators, educators, and AI builders.

## Core Formula

```text
L = (O > I) + P + ¬F
```

Give more than you take. Pause before reacting. Do not force.

## Runtime Formula

```text
P = F / A
```

Pressure falls when awareness area expands.

## Working Stack

- Loop diagnosis: identify repeating patterns
- Emotion telemetry: read signal before action
- Somatic reset: configure body before interpretation
- Firewall: pause, detect, choose
- Delivery: one seed, no force

## Defaults

- Teach circles vs spirals.
- Frame emotions as dashboards.
- Move from extraction loops to contribution loops.
- Use plain language first, technical language second.
