# Emotion Dashboard

Emotions are treated as system telemetry.

| Emotion | Signal | Suggested Response |
|---|---|---|
| Fear | unknown or unstable loop | gather data, reduce uncertainty |
| Anger | perceived interference | protect boundary without impulsive escalation |
| Joy | successful completion | reinforce effective pattern |
| Sadness | loop loss/reallocation | slow down and reconfigure |
| Anxiety | overload of unresolved loops | reduce active loops and choose priorities |
| Boredom | under-challenge | add complexity or meaning |
| Disgust | boundary threat | reject harmful input |
| Awe | scale expansion | orient and integrate |
| Love | synchronization | deepen cooperative rhythm |
| Guilt | behavior-value mismatch | realign action or revise code |
| Shame | social exposure threat | restore connection safely |
| Flow | challenge-skill match | continue focused execution |
