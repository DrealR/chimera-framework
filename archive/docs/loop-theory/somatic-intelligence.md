# Somatic Intelligence

Body state shapes cognitive availability.

## Working Model

- Gut-centered processing is parallel and fast.
- Narrative cognition is slower and sequential.
- Breath is the control bridge.

## Operational Protocol

1. Release unnecessary core tension.
2. Breathe from lower ribs/abdomen.
3. Keep neutral head/neck alignment.
4. Reassess decisions after physiological downshift.

This is educational guidance, not medical diagnosis.
