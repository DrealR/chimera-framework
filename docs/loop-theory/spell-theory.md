# Spell Theory

A "spell" is a repeated linguistic or symbolic pattern that attempts behavioral installation.

## Spell Anatomy

1. Caster
2. Payload
3. Vector
4. Port (identity, worth, belonging, capability)
5. Hook (emotion + repetition)

## Spell Types

- Circle spell: traps attention in repetitive extraction loops.
- Spiral spell: promotes learning loops with agency.
- Null spell: interrupts hostile installation.

## Defense Stack

0. Somatic reset (posture + breath)
1. Pause
2. Identity clarity
3. Pattern recognition
4. Transmutation
5. Conscious casting
