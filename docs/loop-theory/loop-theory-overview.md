# Loop Theory Overview

Loop Theory treats repeating processes as a shared grammar across domains.

## Core Claim

A loop is the smallest unit of persistence: a process that can return and run again.

## Seven Properties

1. Tick rate
2. Nesting depth
3. Feedback type
4. Agency
5. Persistence
6. Anchor
7. Phase (circle vs spiral)

## Practical Interpretation

- Circle loops repeat without learning.
- Spiral loops repeat with adaptation.
- Awareness plus pause is the conversion mechanism.

## Why This Matters

The same diagnostics can be used for personal habits, teams, organizations, and software systems.
