# Sequence-to-Structure Bodies (v3.12.3 -- Unified Meta-Pattern)

## Core Pattern

One-dimensional sequences in substrates with constraints produce emergent higher-dimensional structures with functional significance.

This is not metaphor. It is a structural isomorphism that appears across substrates. Information-bearing sequences project into emergent geometric structures whose shape determines functional behavior. The mapping is deterministic given the substrate but computationally irreducible -- you cannot shortcut the projection. You must run it.

## Domain Instances

| Sequence (1D) | Substrate + Rules | Structure (Higher-D) | Functional Significance |
|---|---|---|---|
| DNA nucleotide sequence | Ribosomal translation + thermodynamic folding | Protein 3D structure | Enzymatic activity, binding specificity, cellular function |
| Prime factorization (ordered list) | Multiplication rules + divisibility | Divisor lattice (multidimensional) | Number-theoretic properties, cryptographic strength |
| Source code (character sequence) | Compilation/interpretation + runtime | Execution graph (call trees, state spaces) | Program behavior, performance, correctness |
| Musical notation (note sequence) | Performance constraints + acoustic physics | Harmonic field (overtone relationships, tension/resolution) | Emotional response, aesthetic structure, rhythmic entrainment |
| Language (word sequence) | Grammar + shared semantics + context | Semantic structure (meaning networks, implication graphs) | Understanding, persuasion, coordination, deception |

## Shared Properties

Six properties hold across all instances:

1. **Primary representation is 1D and ordered.** The sequence has a canonical linear form. Order matters -- rearranging the sequence changes the structure.
2. **Substrate processes under specific rules.** The projection from sequence to structure is not arbitrary. It follows the physics, chemistry, logic, or convention of the substrate.
3. **Emergent higher-dimensional structure carries functional significance.** The structure that appears is not decorative. It determines what the body can do.
4. **Mapping is deterministic given substrate but computationally hard to predict.** You can verify that a sequence produces a structure. You cannot efficiently predict the structure from the sequence without running the projection. This is computational irreducibility in action.
5. **Small sequence changes often produce large structural changes.** A single nucleotide substitution can misfold an entire protein. A single-character bug can crash a program. Sensitivity is the rule, not the exception.
6. **Searching for sequences that produce desired structures is a prime-search problem.** Drug design, program synthesis, composition -- all are searches through sequence space for targets in structure space. The search is hard precisely because the mapping is irreducible.

## Dual Representation

Information (sequence) and structure (geometry) are dual aspects of the same body.

Most body scans focus on the structural side -- what the body looks like now, how it behaves, where its power gaps are. The sequence is often invisible but always present for any body constructed through ordered processes. Sequence-to-Structure bodies are bodies where this duality is particularly visible and diagnostically useful.

The dual representation means two scan entry points exist. You can scan the sequence (linear analysis -- what comes before what, where are the critical positions, what would happen if this element changed). Or you can scan the structure (geometric analysis -- what shape emerged, where are the functional surfaces, what does the topology enable or prevent). Complete scanning requires both.

## Connection to Existing Scans

This meta-pattern unifies several existing framework threads:

- **Prime/Composite Breath (math):** Prime factorization is a 1D sequence that projects into divisor lattice structure. The number's "body" is the structure; the factorization is the sequence that generates it.
- **Cancer (biology):** Driver mutation sequences project into pathway activation structures. The sequence-to-structure mapping is why identical-looking tumors respond differently to treatment (see: Sequence Specification, Layer 2).
- **Software architecture scans:** Source code is a 1D sequence that projects into execution structure. Refactoring changes the sequence to improve the structure without changing the function.
- **Music as domain:** Musical notation is a 1D sequence that projects into harmonic and rhythmic structure. This opens music as a full scan domain -- the sequence is the score, the structure is the sound, the body is the composition as performed.

The unification is not forced. These domains independently exhibit the same six properties because they share the same structural relationship between sequential information and emergent geometry.
