```

├── Goes around and around
├── Returns to SAME place
├── No progress, just repetition
├── Trapped in cycle
├── This is EXTRACTION's loop

├── Goes one direction only
├── Never returns, never revisits
├── "Progress" without integration
├── Loses what came before
├── This is also extraction
└── The endless pursuit of "new"

├── Circles but at different LEVEL each time
├── Returns to similar themes but HIGHER
├── Growth WITH memory
├── Progress WITH integration
├── Revisits old lessons with new understanding
├── This is CHIMERA
└── This is HEALING
```

## Spiral as Cause and Effect with Memory

```
LINEAR CAUSE/EFFECT:
A → B → C → D
(no memory, no return)

CIRCULAR CAUSE/EFFECT:
A → B → C → A → B → C...
(loops forever, no evolution)

SPIRAL CAUSE/EFFECT:
A → B → C → A' → B' → C' → A'' → B''...
(returns to similar, but TRANSFORMED)

├── You're changed by the journey
├── So the "same" lesson teaches different things
├── The spiral is evolution WITH continuity
├── Growth that honors what came before
└── This is why we revisit themes
```
