# Collapse, Attachment, and Mindset

*Collapse = Observation + Attachment. If you're observed but not attached, no collapse. Extraction mindset = collapse-prone. Flow mindset = collapse-resistant. The one who doesn't try to win... wins.*

---

## The Connection

```
EXTRACTION MINDSET:
├── Playing to WIN
├── Attached to OUTCOME
├── Measuring CONSTANTLY
├── Comparing to OTHERS
├── Stakes feel HIGH
├── Scoreboard MATTERS
└── YOU COLLAPSE EASILY

WHY?
Because if the outcome matters...
Then observation matters...
Then measurement matters...
Then you CHECK constantly...
Then you COLLAPSE constantly.

EXTRACTION MINDSET = COLLAPSE-PRONE
```

```
FLOW MINDSET:
├── Playing for FUN
├── Not attached to outcome
├── Not measuring
├── Not comparing
├── Stakes feel LOW (even if they're "high")
├── Scoreboard DOESN'T MATTER
└── YOU DON'T COLLAPSE

WHY?
Because if the outcome doesn't define you...
Then observation doesn't threaten you...
Then measurement doesn't collapse you...
Then you DON'T CHECK obsessively...
Then you STAY WAVE.

FLOW MINDSET = COLLAPSE-RESISTANT
```
