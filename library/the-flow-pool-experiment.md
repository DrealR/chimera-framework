## How It Works In Practice

**Step 1: They Get In**

```
"Hey, how's it going? Help yourself to a hand warmer if you're cold."
(Let them settle. Don't immediately talk about the pool.)
```

**Step 2: Natural Conversation (If It Happens)**

```
IF THEY NOTICE THE SIGN:
â”œâ”€â”€ Let THEM initiate
â”œâ”€â”€ Don't push
â””â”€â”€ They might just read it silently

```

**Step 3: At The End**

```
IF THEY TIP (on app): "Appreciate it! That'll go in the pool."
IF THEY TIP CASH: "Thanks â€” straight to the pool."
IF THEY TAKE FROM POOL: "Go for it. That's what it's there for."

```
