## COHERENCE GAP METRIC

```
CG = |Response_Avg_Normalized - D7_Normalized|

Where:
  Response_Avg_Normalized = (Response_Avg - 1) / 4  (scale 0-1)
  D7_Normalized = (D7 - 1) / 4  (scale 0-1)

LOW GAP (0.0 - 0.2) = COHERENT (words match being)
HIGH GAP (0.5 - 1.0) = INCOHERENT (words don't match being)
```
