## WHAT IS AN AXIOM?

An axiom is a **self-evident truth** that requires no proof. It's the bedrock. Everything else derives from it.

Mathematics has axioms:
- 0 is a natural number (Peano)
- For every set A, there exists a set B such that... (ZFC)

These aren't proved. They're CHOSEN. And from that choice, all of mathematics unfolds.

**The axioms define the God.** Choose different axioms, get a different mathematics. A different system. A different identity.
